### English Version

# 🛒 PyCashier

**PyCashier** is a lightweight virtual point-of-sale (POS) software designed for small businesses. It features a robust product Finder, a streamlined Cashier interface, and an easy-to-use Adder for inventory management. Built for reliability, it runs locally on Windows and Linux without requiring an internet connection.

## ✨ Key Features
* **Product Finder:** Quickly locate items within your local database.
* **Virtual Cashier:** Process transactions seamlessly.
* **Inventory Adder:** Easily add and manage products in your database.
* **100% Offline:** Works anywhere, anytime, ensuring your data stays private.
* **Free & Open Source:** Designed to support growth for small enterprises.

### Indonesian Version

# 🛒 PyCashier

**PyCashier** adalah perangkat lunak kasir virtual ringan yang dirancang khusus untuk mendukung operasional bisnis kecil. Dilengkapi dengan fitur pencarian produk (Finder), antarmuka kasir (Cashier), dan pengelola inventaris (Adder), program ini dapat berjalan secara lokal di Windows dan Linux tanpa memerlukan koneksi internet.

## ✨ Fitur Utama
* **Pencarian Produk:** Temukan item dalam database lokal dengan cepat.
* **Kasir Virtual:** Proses transaksi penjualan dengan antarmuka yang simpel.
* **Penambah Inventaris:** Tambahkan dan kelola produk baru ke dalam database dengan mudah.
* **Akses Offline:** Menjamin privasi data karena tidak memerlukan koneksi internet.
* **Gratis & Open Source:** Solusi teknologi tanpa biaya untuk membantu UMKM.

## ⚠️ Catatan Penting
* **Format Mata Uang:** Program ini dikonfigurasi menggunakan format **Rupiah Indonesia (IDR)**. Jika Anda berada di luar Indonesia, Anda perlu menyesuaikan kode sumber untuk mendukung mata uang wilayah Anda.
* **Latar Belakang Pengembangan:** Proyek ini dikembangkan dengan memanfaatkan **teknologi asisten pengkodean berbasis AI** untuk mengoptimalkan efisiensi pembuatan kode. Meskipun telah diuji fungsionalitasnya, pengguna disarankan untuk meninjau kembali kode sesuai dengan kebutuhan spesifik bisnis.
